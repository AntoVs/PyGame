# Post-COVID Health Data Analysis

## Overview
This project performs **data analysis and visualization** on a dataset related to post-COVID health effects using **Python, Pandas, and Plotly**.

The program analyzes relationships between variables such as age, recovery time, mental health impact, and symptoms. Interactive visualizations help identify patterns in post-COVID health outcomes.

---

## Dataset

**File:** `post_covid_health_effects_dataset.csv`

The dataset includes information such as:

- Age
- Gender
- COVID Severity
- Days to Recovery
- Long COVID Risk
- Mental Health Impact
- Physical Activity Level
- Symptoms (Fatigue, Brain Fog, Breathing Issues, Loss of Taste/Smell)

---

## equirements

- **Python**
- **Pandas** – Data processing and analysis
- **Plotly** – Interactive data visualization

---

## Visualizations

The following plots are generated:

1. **Age Distribution**
2. **Average Recovery Days by COVID Severity**
3. **Recovery Time vs Age**
4. **Long COVID Risk Distribution**
5. **Average Age by Mental Health Impact**
6. **Symptom Correlation Heatmap**

---

## Output

All visualizations are saved in a single HTML dashboard:

plots.html

### Install Dependencies
